Estos codigos fueron sacados de https://grantjenks.com/docs/freegames/ y se modificaron para implementar distintas funcionalidades.
El archivo de JuegoMemoria es un memorama del nombre de distintos paises, en este se implemento un contador de taps, el centrado del texto y un identificador del fin del juego.
El archivo de JuegoPackman es una version del juego de packman, en esta se implemento una generacion de mapa aleatorio, que los fantasmas sigan al usuario y un aumento de velocidad a los fantasmas.
El archivo de JuegoPintado es un juego en el cual eres capaz de utilizar distintos colores y figuras para pintar en un liezo blanco.
El archivo de Juego Tiro es un juego en el cual se disparan pelotas a distintos objativos que se mueven horizontalmente atravez de la pantalla.
El archivo de Juego Vibora es una representacion del juego de snake, en esta se implemento un aumento de velocidad cuando se comen las frutas, el movimiento aleatorio de las mismas y el cambio de colores aleatorio de la seerpiente y la fruta.

Requerimientos:
Tener python installado
Tener el pip installado
Tener la libreria de freegames
